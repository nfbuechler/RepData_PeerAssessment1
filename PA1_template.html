-Devise a strategy for filling in all of the missing values in the dataset. The strategy does not need to be sophisticated. For example, you could use the mean/median for that day, or the mean for that 5-minute interval, etc. I am going to use the mean for that 5-minute interval

myfun <- function(x) {
  steps <- x[,"steps"] #separates out sort column
  rest_data <- x[,c(2,3)] #from rest of data
  steps[which(is.na(steps)==TRUE)] = mean(steps,na.rm = T) #replaces NA values with mean of steps column
  cbind(rest_data, steps)  #binds back together
}
